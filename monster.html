2D Monster Game

This was a game created my freshman year, it is an intricate in termial 2D game that uses a map that you move around on and try to survive as long as you can without losing all your HP

Using C++ I was able to make different elements to make one cohesive game. It is a mix between pokemon and Witcher since when you move you have a random chance to fight a random monster. From there you are given choices on what you want to do. You also have an inventory system and a upgrade system.
